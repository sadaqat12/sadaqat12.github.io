Top Stations of Los Angeles!

One of the objectives was to find out which stations were the busiest starting points and which were the busiest ending points.

Busiest Start Station: 3069

Busiest Ending Station: 3005

Using Tableau I was able to find that the busiest station for starting off a trip was Station 3069. While the busiest for ending was Station 3005. I excluded station 4108 from both of these charts. This exclusion does not affect my intended purpose much which was to find the busiest stations, and station 4108 was pretty slow. Then I located both of the stations using the given coordinates, by plugging the latitude/longitude pairs into Google Maps.
